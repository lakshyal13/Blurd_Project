# 🌐 BLURD – Neurodivergent Community Platform

BLURD is a cloud-based community web application designed for ADHD and dyslexic users to communicate, share thoughts, and interact in a simple and supportive environment.

## 🚀 Live Demo

🔗 https://lakshyal13.github.io/Blurd_Project/Frontend/index.html

## ✨ Features

* 💬 Multi-user chat system
* ☁️ Cloud-based backend (accessible from anywhere)
* 📊 Mood detection using sentiment analysis
* 🧠 Simple and distraction-free UI
* 🔄 Real-time-like updates using polling

## 🏗️ Tech Stack

### 🌐 Web Technology

* HTML, CSS, JavaScript
* Node.js & Express.js
* REST APIs (GET/POST)

### ☁️ Cloud Computing

* Backend deployed on Render
* Database hosted on MongoDB Atlas
* Client-server architecture

### 📊 Data Science

* Rule-based sentiment analysis
* Keyword-based mood classification (positive / negative / neutral)

## ⚙️ How It Works

1. User sends a message
2. Frontend sends request to backend API
3. Backend processes message and analyzes mood
4. Data is stored in MongoDB Atlas
5. Messages are fetched and displayed to all users

## 🧪 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/lakshyal13/Blurd_Project.git
```

2. Go to backend:

```
cd backend
npm install
node server.js
```

3. Open frontend:

```
frontend/chat.html
```
